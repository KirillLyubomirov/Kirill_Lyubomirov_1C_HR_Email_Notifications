# Тест-план

Объект тестирования: учебная конфигурация **«Кадровые уведомления 1С:ЗУП»**.

Цель тестирования: проверить создание справочных данных, проведение кадровых документов, регистрацию движений, формирование отчётов и наличие бизнес-процесса.

## Тестовое окружение

- Платформа: 1С:Предприятие 8.3.
- Тип базы: файловая информационная база.
- Режим уведомлений: тестовая регистрация отправки без реального SMTP-подключения.
- Основной пользователь: `admin01`.

Конфигурация прошла функциональную проверку. Основной сценарий регистрации кадрового изменения и фиксации уведомлений выполняется. Отчёты формируются на основании тестовых данных. Бизнес-процесс создан и содержит карту маршрута обработки кадрового изменения.
